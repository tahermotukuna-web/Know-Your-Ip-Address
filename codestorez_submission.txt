# Know Your IP Address — Quick IP Checker for Developers & Admins

Know Your IP Address is a polished, lightweight PHP application that displays a visitor's public IPv4/IPv6 address along with connection diagnostics (proxy detection, validation, and type). This product is packaged and ready to deploy for testing, diagnostics, tutorials, or embedding in dashboards.

Key benefits for buyers:
- Ready-to-deploy — works on XAMPP, Apache, or any PHP hosting without composer
- Easy to brand and customize — simple layout and CSS files included
- Minimal footprint — fast, secure, and mobile friendly

Highlights
- Display public IPv4/IPv6
- Proxy detection (HTTP_CLIENT_IP / HTTP_X_FORWARDED_FOR)
- IP validation and detection (IPv4 or IPv6)
- Copy-to-clipboard UX for quick use
- Clean, responsive design with accessible markup

What's included (download package)
- Source code (index.php, `src/` controllers/services/helpers)
- UI assets (CSS and JS in `public/`)
- Example `.htaccess` for Apache
- Marketplace-ready documentation and licenses (`codestorez_submission.txt`, `FREEMIUM_LICENSE.txt`, `LICENSE`)

Who it’s for
- IT administrators, network engineers, and system administrators
- Devs and instructors who want an embeddable IP checker
- SaaS owners who want an IP-check widget to add to dashboards

Requirements
- PHP 7.4 or later (recommended PHP 8.x)
- Apache or compatible web server (XAMPP recommended for local testing)
- Basic write access to a webroot (htdocs or virtual host)

Installation (Marketplace buyer-friendly)
1. Upload the files to your server’s webroot (or a subfolder).
2. Ensure permissions are correct and PHP is configured (no Composer required).
3. Access the site in your browser (e.g., `https://example.com/index.php`) and verify the IP display.

Quick Setup (XAMPP example)
```bash
# download and unzip the package to C:\xampp\htdocs\Know-Your-Ip-Address
# start XAMPP and make sure Apache is running
# visit http://localhost/Know-Your-Ip-Address/index.php
```
